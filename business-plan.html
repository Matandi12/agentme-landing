📌 1. סיכום מנהלים

שירות התקנה וניהול של סוכני AI אישיים בוואטסאפ - עוזרים חכמים שעובדים 24/7, עונים בשם הלקוח, מנהלים משימות, ועוזרים בכל תחום.

💡 הרעיון בקצרה

אתה מתקין ללקוחות סוכן AI כמו "לוגן" - עוזר חכם שחי בוואטסאפ שלהם, לומד את הסגנון שלהם, ועוזר להם בכל דבר. הלקוח משלם מנוי חודשי, אתה מרוויח את ההפרש בין המחיר לעלויות.

📱 2. WhatsApp: Baileys vs Business API

יש שתי דרכים לחבר סוכן AI לוואטסאפ. הנה ההבדלים:

✅ Baileys (מה שאתה משתמש)

  • עלות: חינם לגמרי
  • התקנה: פשוטה, סריקת QR
  • יציבות: טובה מאוד
  • סיכון: קטן - אם WhatsApp יזהה "התנהגות חשודה" יכולים לחסום את המספר
  • מתאים ל: רוב הלקוחות, שימוש סביר

המלצה: להתחיל עם זה!

🏢 WhatsApp Business API (רשמי)

  • עלות: $15-50/חודש + תשלום לפי הודעות
  • התקנה: מורכבת, דרך Meta Business
  • יציבות: מושלמת
  • סיכון: אפס - פתרון רשמי
  • מתאים ל: עסקים גדולים, נפח הודעות גבוה

לשקול בעתיד לחבילת פרימיום

🤔 מתי עוברים ל-Business API?

אם לקוח שולח מאות הודעות ביום, או שיש לו עסק גדול שלא יכול להרשות לעצמו סיכון של חסימה - אז כדאי להציע לו חבילת פרימיום עם Business API. לרוב הלקוחות, Baileys מספיק בהחלט.

🛠️ 6. מדריך התקנה שלב-אחר-שלב

ככה מתקינים סוכן AI חדש ללקוח:

1 יצירת שרת EC2

נכנסים ל-AWS Console ויוצרים instance חדש:

  1. לוחצים "Launch Instance"
  2. בוחרים Ubuntu 22.04 או 24.04
  3. בוחרים t3.micro או t3.small
  4. יוצרים Key Pair חדש (לשמור את הקובץ!)
  5. Security Group: לפתוח פורט 22 (SSH)
  6. לוחצים Launch

זמן: 5 דקות

2 התחברות לשרת

ssh -i "customer-key.pem" ubuntu@[IP-ADDRESS]

מחליפים את [IP-ADDRESS] בכתובת ה-IP של השרת החדש.

3 התקנת Node.js

# עדכון המערכת
sudo apt update && sudo apt upgrade -y

# התקנת Node.js 22
curl -fsSL https://deb.nodesource.com/setup_22.x | sudo -E bash -
sudo apt install -y nodejs

# בדיקה שזה עובד
node --version
npm --version

זמן: 3 דקות

4 התקנת OpenClaw

# התקנת OpenClaw גלובלית
sudo npm install -g openclaw

# בדיקה
openclaw --version

זמן: 2 דקות

5 הגדרת OpenClaw

# יצירת תיקיית עבודה
mkdir -p ~/.openclaw/workspace
cd ~/.openclaw

# הרצת ההגדרה
openclaw init

במהלך ההגדרה:

  • להזין API Key של Anthropic
  • לבחור מודל (Claude Sonnet מומלץ)
  • להגדיר שם לסוכן

זמן: 5 דקות

6 חיבור WhatsApp

# הוספת ערוץ וואטסאפ
openclaw channels add whatsapp

יופיע QR Code - הלקוח סורק אותו עם וואטסאפ שלו:

  1. פותח וואטסאפ בטלפון
  2. הגדרות → מכשירים מקושרים
  3. קישור מכשיר → סורק את ה-QR

זמן: 2 דקות

7 הפעלת הסוכן

# הפעלה ברקע
openclaw gateway start

# בדיקת סטטוס
openclaw gateway status

הסוכן עכשיו חי ועונה להודעות! 🎉

8 הגדרת הפעלה אוטומטית

# יצירת service שרץ אוטומטית
sudo nano /etc/systemd/system/openclaw.service

תוכן הקובץ:

[Unit]
Description=OpenClaw AI Agent
After=network.target

[Service]
Type=simple
User=ubuntu
WorkingDirectory=/home/ubuntu/.openclaw
ExecStart=/usr/bin/openclaw gateway start --foreground
Restart=always

[Install]
WantedBy=multi-user.target
# הפעלה
sudo systemctl enable openclaw
sudo systemctl start openclaw

עכשיו הסוכן יפעל אוטומטית גם אחרי ריסטרט של השרת.

⏱️ זמן התקנה כולל: 30-45 דקות

אחרי שתעשה את זה כמה פעמים, זה ייקח 15-20 דקות בלבד.

🎨 7. התאמה אישית לכל לקוח

הגדרת אופי הסוכן

יוצרים קובץ SOUL.md בתיקיית workspace:

# SOUL.md - האופי של הסוכן

## שם
[שם שהלקוח בוחר]

## סגנון דיבור
- רשמי / לא רשמי
- עברית / אנגלית / שילוב
- קצר וענייני / מפורט

## התמחות
- [תחום העסק של הלקוח]
- [משימות עיקריות]

## כללים
- מה לעשות אוטומטית
- מתי לשאול לפני פעולה
- מה אסור לעשות

חיבור אינטגרציות (חבילת עסקי)

  • Google Calendar: תיאום פגישות אוטומטי
  • Google Sheets: ניהול לידים ונתונים
  • Email: שליחת וקריאת מיילים

הדרכת הלקוח

  1. הסבר מה הסוכן יכול לעשות
  2. דוגמאות לשימוש
  3. איך לתת הוראות לסוכן
  4. מה לעשות אם משהו לא עובד
